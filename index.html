ThaiAirways X Globaleur Tour Booking Service

Globaleur's first project with ThaiAirways (second airline customer behind Royal Air Maroc), a tours and activity booking serivce. The service allows users to book tours, activities and even sim cards for their upcoming travels. It is also fully locallized in English and Thai to give ThaiAirways customers a better experience.
A link to the site can be found on ThaiAirways' website under the "More Services" section.

This project was built using the same codebase as the KBCard project. Since the frontend was designed to be a whitelabel, we were able to create a new "client" and enable/disable features to get this service to work.

Technology

The front-end was built using ReactJS and BootstrapCSS, and designed to be a fully customizable whitelabel website. The backend was done in NodeJS and Golang, and deployed to Azure's AppService and Frontdoor services.

Mobirum Discovery

A gaming news portal brining you the latest information about the asian gaming industry. With articles, videos and forums, Mobirum Discovery has you covered!

Technology

The UI for Mobirum Discovery was designed from scratch, utilizing Bootstrap and custom handwritten CSS. ReactJS was used to build the front-end using server-side rendering to boost page loading. To help boost SEO, JSON-LD was used for every page to help crawlers better understand the data and structure of the page.

For the backend, a NodeJS server utilizing ExpressJS sitting behind an NGINX proxy was used. A MongoDB database was used for storing all site data, while a Redis database was used to cache frequently used data.
